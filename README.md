# opencv46_mingwmake_by_GCC10_3
windows使用TDM-GCC10.3  cmake, mingw32-make  opencv4.6 64-bit基本軟體
整個就編成單一一個dll檔：
libopencv_world460.dll，
方便vscode, dev C++等IDE使用
解壓縮就能用

重要檔案資料夾結構:
install
---include
------opencv2
---x64
------mingw
---------bin
---------lib
